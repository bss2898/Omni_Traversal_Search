# Omni_Traversal_Search
TEM Search is a multi-pointer linear search strategy that inspects:
- Left edge
- Right edge
- Middle
- Middle-left
- Middle-right
simultaneously.

This repository benchmarks TEM Search against:
- Linear Search
- Bidirectional Search
under multiple data distributions.

## Current Status
Research experiment only.
No claim of asymptotic improvement over O(n).
Focus is on practical performance under biased target distributions.
